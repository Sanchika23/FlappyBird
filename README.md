# 🐦 Flappy Bird Clone

A smooth, modern take on the classic Flappy Bird game built with dynamic difficulty and a clean visual style.

## 🚀 Features
- **Smooth Physics**: Fluid Gravity** and flapping mechanics.
- **Dynamic Difficulty**: The game gets harder every 15 points (smaller gaps and faster pipes).
- **Persistence**: High score is saved automatically using local storage.
- **Responsive Controls**: Play using the **Space bar** or **Mouse Click**.
- **Modern Visuals**: 16:9 aspect ratio with rounded corners and smooth animations.

## 🛠️ Tech Stack
- HTML5 Canvas
- CSS3
- Vanilla JavaScript

## 🎮 How to Play
1. Clone the repository or download the files.
2. Open `index.html` in any modern web browser.
3. Press **Space** or **Click** to start and keep the bird in the air!
